# Images Directory

Place your game assets here:

## Required Images

- **crystal-placeholder.png** (or .jpg) - Main crystal image shown on the home screen
  - Recommended size: 400x400px or larger
  - Should have transparent background or match dark theme

## Prize Images (Optional)

- **scared_cat.png** - For "Scared Cat" prize
- **skull_flow.png** - For "Skull Flow" prize
- Other prize images as needed

## Icon Assets (Optional)

- Currency icons (diamond, bat, etc.)
- Navigation icons
- Task icons

## Notes

- All images should be optimized for web (use WebP or compressed PNG/JPG)
- Consider providing @2x versions for retina displays
- Update image paths in templates and seed data script after adding images
